      OpenType
    • TTF - TrueType квадратичные кривые Безье (100 кб)
    • OTF - PostScript кубические кривые (100 кб)
    • WOFF - незащищенный архив исходника OTF или TTF пожалуй, самый важный формат, который поддерживают большинство популярных браузеров, а файлы в WOFF обычно 2–2,5 раза легче, чем исходные; (50кб)
    • EOT - внедряемый TT OpenType архив, имеющий механизмы защиты, нужен для поддержки старых браузеров Internet Explorer (начиная с IE8 кроме TrueType кривых, поддерживаются и PostScript);(50кб)
    • SVG — для поддержки браузера Safari.
  • -webkit-font-smoothing - способ растеризации css

    • regular - нормальный
    • italic - курсивный
