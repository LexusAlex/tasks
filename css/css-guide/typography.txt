2. Нужно искать разборчивый шрифт с большим простансвом и четкими штризами
3. Текст verdana легко читается и более разборчивый
4. Как выбрать шрифт для передачи смысла
    Gergia - это переходный шрифт
    Serif 72
    Museo
    Arial - без засечек
    Museo sans - сила и стабильность

    italic - кричащий более элегантный
    Comic sans - шрифт комикса
    Courier - печатная машинка

    Serif - хорош для заголовков
    Выбрать шрифт который мог бы работать
    Не раздражает читателей
