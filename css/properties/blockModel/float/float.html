Плавающая система позиционирования

  1. Бокс выпадает из нормального потока
  2. Бокс полностью игнорируется блочными боксами
  3. Инлайн боксы обтекают плавающий бокс
  4. Прижимается к краю родителя
  5. Можно располагать в любом месте документа
  6. Float определяет, по какой стороне будет выравниваться элемент
  7. Плавающие (обтекающие) элементы сначала выстраиваются в нормальном потоке, затем образуется новый поток, иони сдвигаются либо вправо, либо влево
  8. Как правило, плавающий элемент обязательно должен иметь фиксированную ширину.
  9. Используя свойство clear
  10. В своем HTML коде, я почти всегда сначала создаю плавающие элементы во время разметки, прежде чем добавлять простые элементы, которые могут взаимодействовать с ними. Вы экономите большую часть времени, и это дает желаемый результат.
  11. Любой плавающий элемент будет находится либо рядом, либо ниже предыдущего элемента.
  12. Плавающий элемент не может быть выше своего соседа плавающего элемента.
  13. Плавающий элемент не может быть выше своего соседа строчного элемента.
  14. Плавающий элемент должен быть помещен как можно выше.
  15. Браузер пытается отобразить блоки так, чтобы left и right не пересекались
